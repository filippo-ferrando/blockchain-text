# Blockchain-based text sharing



### To run the application type in your terminal:

```python
export FLASK_APP=node_server.py
flask run --port 8000
```

### In another window type

```python
python run_app.py
```



#### If you want use multiple node then use:

```python
flask run --port 8000
flask run --port 8001
flask run --port 8002
ecc.
```
